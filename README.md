## **🛠️ Installation**

1. **Clone the repository**

   ```sh
   git clone https://github.com/your-username/your-repo.git
   cd your-repo
   ```

2. **Install dependencies**

   ```sh
   npm install
   ```

3. **Start the development server**

   ```sh
   npm run dev
   ```

4. **Build for production**

   ```sh
   npm run build
   ```

5. **Run tests**

   ```sh
   npm run test
   ```
